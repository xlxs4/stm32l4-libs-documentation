__STATIC_INLINE void LL_EXTI_DisableRisingTrig_0_31 ( uint32_t  ExtiLine)

Disable ExtiLine Rising Edge Trigger for Lines in range 0 to 31.

Note
The configurable wakeup lines are edge-triggered. No glitch must be generated on these lines. If a rising edge on a configurable interrupt line occurs during a write operation in the EXTI_RTSR register, the pending bit is not set. Rising and falling edge triggers can be set for the same interrupt line. In this case, both generate a trigger condition. @rmtoll RTSR1 RTx LL_EXTI_DisableRisingTrig_0_31
